animation-timing-function Specifies how the animation makes transitions through keyframes. It can have the following values: ease: The animation starts slowly, then fast, and then finally ends slowly (this is the default) linear: The animation plays with the same speed from start to end ease-in: The animation plays with a slow start ease-out: The animation plays with a slow end
